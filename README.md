# Little OS

A tiny little x86 (IA-32) operating system inspired by [The little book about OS development](https://littleosbook.github.io/) and [OSDev Wiki](http://wiki.osdev.org/Main_Page).

### Done

* Framebuffer
* Serial Port
* Segmentation
* Input from the Keyboard (Interrupts)

### Current 

* Paging
